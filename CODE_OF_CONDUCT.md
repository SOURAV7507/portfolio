# Code of Conduct

## 1. Purpose
This Code of Conduct ensures a respectful, safe, and welcoming environment for everyone.

## 2. Expected Behavior
- Be respectful and supportive.
- Use welcoming and inclusive language.
- Accept constructive criticism gracefully.

## 3. Unacceptable Behavior
- Harassment, discrimination, insults.
- Personal attacks or trolling.
- Sharing private information without permission.

## 4. Enforcement
Project maintainers may take action including:
- Warning
- Temporary or permanent ban
- Removal from project participation

## 5. Reporting Issues
If you observe unacceptable behavior, report it by opening an issue in the repository.

Thank you for helping maintain a healthy project environment!
